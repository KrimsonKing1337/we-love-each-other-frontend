# we-love-each-other-frontend
Frontend for we love each other project
(it's just a widget though)

### Install

```
npm i -g stylelint prettier eslint lint-staged
npm i
```

### Start webpack dev-server
```
npm run start
```

or

```
npm run start:dev
```

if you need development enviroment.

### Make build
```
npm run build
```

or

```
npm run build:dev
```

if you need development enviroment
